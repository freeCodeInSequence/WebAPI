API stands for Application Programming Inerface.

Web API is a concept (not a technology)that works on HTTP protocol and 
it is used to extend the functionality of an application.

How to create the Web API
We can develop Web APIs using multiple Technologies such as Java, .Net, node js etc.
Each technology provides couple of frameworks to develop the Web API.

Create the Web API using .Net

In .Net we have following two frameworks to create the Web API
	Asp.Net WebAPI 2 (using .Net framework)
	Asp.Net Core (using .Net core)
